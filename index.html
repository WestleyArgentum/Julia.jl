Julya

Curated decibans of Julia language.

Download this project as a .zip file Download this project as a tar.gz file

http://svaksha.github.io/julya contains curated deciban's of Julia language learning resources.

It is mirrored on BB: https://bitbucket.org/svaksha/julya.

Join the http://julialang.org/community/ or head over to http://webchat.freenode.net/?channels=julia for a live chat.
