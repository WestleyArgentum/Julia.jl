Julia.jl

Curated decibans of Julia language.

The svaksha.github.io/julya contains curated decibans of Julia language learning resources, mirrored at BB.

PAGES

  • Actuarial-Science.md :: Finance, economics, stochastic, insurance Statistics... packages.
  • Algorithms.md :: AI, DataMining, Data Structures, ML, NLP, ...
  • API.md :: API's for C++, Fortran, Go, Java, JavaScript, MATLAB, Perl, Python, R, ...
  • Benchmarks-Optimization.md :: Benchmarks and Optimization toolkits.
  • Build-Automation.md :: Tools for continuous integration (CI), continuous delivery (CD), Packaging, release engineering (RE), release management (RM), software configuration management (SCM), etc...
  • Computer-Graphics.md :: Plotting, Graphics and other Visualization tools.
  • Cookbook.md :: blogs, cookbooks, cheatsheets, IJulia NoteBooks, etc...
  • Database.md :: NoSQL, RDBMS and Middleware API's.
  • Distributed-Computing.md :: Distributed Computing, Networking, Servers,...
  • i18n-L10n.md :: Transliteration, Internationalisation (i18n) and Localisation (L10n)
  • Mathematics.md:: Algebra, Geometry,... anything Math related.
  • NEWS.md
  • Programming-Concepts.md :: Programming language concepts that are used in the type system, data types, etc..
  • QA.md :: Test Driven Development, Sandbox, Functional/ Unit testing,... Quality-related tools.
  • Science.md :: Astronomy, Biology, Chemistry, GIS, Meteorology, Physics,..
  • UI-UX.md :: User Interface software in Julia.
  • Utilities.md :: Handy toolkits for your Desktop and other general utilities.
  • X-Platform-SW.md :: Cross-Platform Software and API for Hardware arch. (ARM, MIPS, GPU, CUDA), language API, HPC, and other libraries.

NB: As a new language in the scientific computing scene, new libraries are constantly being added, so there will be frequent changes and page reordering. The Julya repo only provides a list (of links) of Julia packages out in the wild, its not an endorsment of the software quality/ features, etc.. For Base packages, check if the package you seek is listed in the built-in package manager on github or check the metadata for registered Julia packages

Join the http://julialang.org/community/ or head over to #julia on irc.freenode.net for a live chat.

Contributing

Some tips for submitting a pull request (PR):

  • Add your link as per the topic, with notes (if any) to the respective MarkDown (.md) pages in the repo.
  • For broken links submit a bug report, or replace the broken URI, then submit a PR.
  • For Documentation and cookbooks, check if it matches the categories listed, if yes, then list it against DOC, else use the Cookbook.md page.
